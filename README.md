# Two Dimensional Fourier Transform  

## 1.手順説明  
- 画像を一度フーリエ変換して、それぞれの点が正弦波を表す画像スペクトルに変換する。その後、そのスペクトルをもとに正弦波を重ね合わせてもとに戻す逆フーリエ変換を行う。  

## 3.結果  
- 少し荒くなってしまったが、求める画像が出てきた。
